# YouTube Preview Speed

A lightweight Chrome extension that lets you control the playback speed of **YouTube's inline hover previews**.

Instead of watching hover previews at the default speed, you can choose between **0.5×**, **1×**, **1.5×**, and **2×** directly from a native-looking dropdown integrated into the preview controls.

## Features

* 🎬 Change hover preview playback speed
* ⚡ Instant speed switching
* 💾 Remembers your preferred speed
* 🎯 Works only on YouTube hover previews (does not affect the main player)
* 🪶 Lightweight — no frameworks or build tools
* 🧩 Manifest V3

## Motivation

While YouTube allows previews to autoplay when hovering over thumbnails, it doesn't provide any way to control their playback speed. This extension aims to add that missing functionality while feeling like a built-in YouTube feature.

## Tech Stack

* Vanilla JavaScript
* Manifest V3
* Chrome Storage API
* CSS
